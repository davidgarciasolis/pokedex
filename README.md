# ⚡ PokéDex Ultimate

Una aplicación web moderna, rápida y visualmente impactante para explorar el mundo Pokémon. Construida con tecnologías web puras (HTML5, CSS3, JavaScript) y alimentada por la potente [PokéAPI](https://pokeapi.co/).

![Preview](https://raw.githubusercontent.com/PokeAPI/sprites/master/sprites/pokemon/other/official-artwork/25.png)

## ✨ Características Principales

- **🔍 Búsqueda Inteligente**: Encuentra Pokémon por nombre parcial o por su número de ID de forma instantánea.
- **🏷️ Filtrado Avanzado**: Combina múltiples tipos para encontrar exactamente el Pokémon que buscas (ej. Agua + Volador).
- **📱 Diseño Responsive**: Experiencia optimizada para móviles, tablets y escritorio.
- **💎 Estética Premium**: Interfaz moderna basada en *glassmorphism*, gradientes vibrantes y micro-animaciones fluidas.
- **📊 Datos en Tiempo Real**: Estadísticas base, habilidades, peso, altura y tipos actualizados.
- **🚀 Rendimiento Optimizado**: Implementación de *debouncing* en búsquedas y carga infinita de resultados.

## 🛠️ Tecnologías Utilizadas

- **HTML5**: Estructura semántica y optimizada para SEO.
- **CSS3 (Vanilla)**: Diseño personalizado sin frameworks, utilizando variables CSS y animaciones complejas.
- **JavaScript (ES6+)**: Lógica asíncrona para consumo de API y manipulación dinámica del DOM.
- **Google Fonts**: Tipografías modernas (*Outfit* y *Space Grotesk*).

## 🚀 Instalación y Uso

No requiere compilación ni dependencias externas. Solo necesitas un navegador web moderno.

1. Clona este repositorio:
   ```bash
   git clone https://github.com/tu-usuario/pokedex.git
   ```
2. Abre el archivo `index.html` en tu navegador favorito:
   ```bash
   open index.html
   ```

## 📸 Capturas de Pantalla

> [!TIP]
> Haz clic en cualquier tarjeta de Pokémon para abrir una vista detallada con todas sus estadísticas base.

- **Filtros**: Panel desplegable con chips interactivos para cada tipo.
